▶ 변수의 선언
		자료형 변수명 = 초기값;
		자료형 변수명;

		변수 선언시 RAM
			int x = 10;

			x 라는 이름의 저장공간이 RAM에 할당되고, 10이라는 값이 들어간다.
			공간이 만들어졌으므로 주소값이 부여되지만 알 수 없다.
			그렇기 때문에 무조건 이름으로 찾아가야 한다.

▶ 변수 선언시 주의사항
		같은 이름의 변수를 여러개 선언할 수 없다.
		초기화를 해준다.
		되도록 선언부(메소드 상단부)에 한꺼번에 선언한다.

▶ 변수명 주의사항
		숫자는 맨 앞에 쓸 수 없다.(1num -> num1)
		예약어는 쓸 수 없다.
		공백을 쓸 수 없다.
		특수문자를 쓸 수 없다.
		_(언더바)는 사용 가능하다.(띄어쓰기 용으로)(no_Where)
		카멜 표기법 : noWhere(소문자 대문자 왔다갔다)

		되도록 소문자만 사용한다.(첫글자는 무조건 소문자!!!)
		되도록 한글은 사용하지 않는다.
		뜻이 있는 명사형 단어를 사용한다.
		(a, b, c, ...) X	(num, height, data, ...) O

▶ 변수의 사용
		변수명을 그대로 사용한다.

		int data = 10;

		System.out.println(data);		// data라는 공간이 가지고 있는 값 사용
		data = 5;					// data라는 공간을 사용
		data = data + 5;		// data가 가지고 있던 값 + 5 해서 data변수에 다시 넣어라!

▶ 변수의 사용 이유
		1. 반복되는 같은 의미의 값을 쉽게 관리하기 위해서
		2. 값에 의미부여를 하기 위해서

▶ 서식문자(format 문자)
		따옴표 안에 존재한다.
		% 와 함께한다.

		%d : decimal(10진수 정수)
		%c : char(문자)
		%f : float(실수)
		%s : string(문자열)
		%% : %

	printf("서식문자열", 값1, 값2,...)
		서식문자열은 서식문자가 포함된 문자열
		ex)
		"%d>%c" ----> 정수 한개 뒤에 '>'가 있고 그 바로 뒤에 문자 한개 있는 문자열

		뒤에 올 값들은 앞의 서식문자열에 있는 서식문자의 개수와 타입에 맞게 따라와야 한다.

		printf("이름 : %s\n", "홍길동");
		println("이름 : 홍길동")

		차이 : 이름만 바꾸고 싶다면 printf는 "홍길동"이 분리되어 있기 때문에
			홍길동만 바꾸면 되지만 println은 다 바꿔야 됨
